# 快速參考卡片

## 交易前檢查清單

### ✓ 每筆交易前必檢查

```
□ 均線排列清晰？(不能混亂)
□ 進場點在支撐/阻力附近？
□ 成交量明顯放大？(>平均150%)
□ MACD/RSI信號清晰？
□ 前一個交易結果？(連虧>2筆則停止)
□ 市場流動性充足？(小幣種避免)
```

## 核心參數速查表

| 指標 | 參數 | 說明 |
|------|------|------|
| MA快 | 5日 | 不要改 |
| MA中 | 20日 | 不要改 |
| MA慢 | 60日 | 不要改 |
| MACD | 12,26,9 | 標準參數 |
| RSI | 14期 | 標準參數 |
| ATR | 14期 | 波動率 |
| 止損倍數 | 2.0倍ATR | 風險額計算用 |
| 止盈倍數 | 3.0倍ATR | 維持1:1.5比例 |
| 最小成交量 | 1.5倍平均 | 突破真假判斷 |
| 單筆風險% | 2% | $1000帳戶=$20 |

## 訊號判斷

### 買入信號 (多頭)

✓ **必須同時滿足**
- [ ] 5MA > 20MA > 60MA (多頭排列)
- [ ] MACD金叉 或 RSI穿越50向上
- [ ] 成交量 > 20根MA × 1.5
- [ ] K線形態優良(錘子線/吞沒線)

### 賣出信號 (空頭)

✓ **必須同時滿足**
- [ ] 5MA < 20MA < 60MA (空頭排列)
- [ ] MACD死叉 或 RSI穿越50向下
- [ ] 成交量 > 20根MA × 1.5
- [ ] K線形態優良(倒錘子線/吞沒線)

## 止損止盈計算

```
步驟1：查看ATR(14) 當前值 = X

步驟2：計算止損距離
       止損距離 = X × 2.0
       
步驟3：計算止盈距離
       止盈距離 = X × 3.0
       
步驟4：確認風險損益比
       止盈距離 ÷ 止損距離 = 1.5 ✓

步驟5：進場
       買入價 = 當前價
       止損點 = 當前價 - 止損距離
       止盈點 = 當前價 + 止盈距離
```

## 快速診斷

### 勝率低 (<50%)

❌ 問題：信號太激進或進入盤整期

✓ 對症下藥：
1. 提升最小成交量要求到2.0倍
2. 檢查ATR比值 (<0.8時盤整)
3. 加入背離確認
4. 等待MA重新排列

### 連續虧損 (>3筆)

❌ 問題：市場環境改變或參數失效

✓ 對症下藥：
1. 立即降低倉位到0.5%風險
2. 檢查是否進入盤整期
3. 暫停交易24小時評估
4. 用最保守參數重啟

### 信號太稀少 (<5筆/周)

❌ 問題：過濾條件太嚴格

✓ 對症下藥：
1. 降低最小成交量到1.3倍
2. 關閉背離確認
3. 放寬K線形態要求
4. 檢查是否該交易對流動性不足

## K線型態速認

### 上漲信號

| 型態 | 特徵 | 可靠度 |
|------|------|--------|
| 錘子線(Hammer) | 下長影、實體小、上影短 | ⭐⭐⭐⭐ |

## 成交量訊號

### 量能放大 (最強訊號)

```
成交量比值 = 當前成交量 / 20根MA平均

比值 > 3.0  → 成交量激增 → 突破通常真實
比值 2.0-3.0 → 明顯放大 → 安全
比值 1.5-2.0 → 正常放大 → 可交易
比值 < 1.5   → 量不足 → 避免交易
```

### 量能警告

❌ 避免交易場景：
- 下跌時成交量萎縮(弱勢)
- 上漲時成交量萎縮(假突破)
- 成交量平緩(無熱度)
- 小幣種流動性差

## 市場環境判斷

### 趨勢行情 (30% 的時間)

✓ 特徵：
- MA明確排列(5>20>60或相反)
- ATR持續升高
- 價格不回踩MA
- 連續新高/新低

✓ 策略：
- 順勢而為
- 逐步加倉
- 動態止損(棘輪)

### 盤整行情 (70% 的時間)

❌ 特徵：
- MA糾纏(5與20/60混在一起)
- ATR降低
- 價格在區間反覆
- 時不時觸及支撐/阻力

❌ 策略：
- 降低倉位到0.5%
- 或完全避免交易
- 等待MA再次排列

## 緊急停損條件

### 出現時立即平倉

1. ❌ 連續虧損達3筆
2. ❌ 月累計虧損 > 3%
3. ❌ ATR快速飆升(可能黑天鵝)
4. ❌ 重要新聞發布(政策/監管/安全事件)
5. ❌ 市場出現十字星群(方向不明)
6. ❌ 流動性突然枯竭(幣安無交易量)

## 交易紀律三原則

### 原則1：紀律執行

✓ DO：
- 按計畫進場
- 按設置止損止盈
- 記錄每筆交易理由

❌ DON'T：
- 看著虧損想翻身加倉
- 隨意改變止損止盈
- 因為情緒改變交易

### 原則2：風險優先

✓ DO：
- 先設止損再看利潤
- 單筆風險不超2%
- 連虧立即停止

❌ DON'T：
- 為了更大利潤放寬止損
- 算不清楚就開倉
- 賭博式交易

### 原則3：耐心等待

✓ DO：
- 等待三層都確認
- 寧願漏掉機會也不追
- 定期評估策略

❌ DON'T：
- 信號不足就開倉
- 追高殺低
- 每天必須有交易

## 每日交易計劃

### 開盤前(10分鐘)

```
□ 檢查4小時趨勢(多/空/盤整)
□ 檢查1小時MA排列
□ 設定15分鐘看盤區間
□ 檢查今日新聞/事件
□ 確認帳戶資金與止損點
```

### 交易中(實時)

```
□ 只在第一層(趨勢)確認才看後續
□ 信號出現才考慮進場
□ 進場立即設止損止盈
□ 記錄進場理由
□ 不要盯著虧損的單子
```

### 收盤後(15分鐘)

```
□ 記錄今日交易
  - 進場點
  - 出場點
  - 理由
  - 執行質量
□ 統計
  - 今日勝率
  - 夏普比變化
□ 明日計畫
```

## 月度回顧

### 每月末檢查這些指標

| 指標 | 目標 | 實績 | 判斷 |
|------|------|------|------|
| 總交易數 | >15筆 | ___ | 信號充足? |
| 勝率 | 55-65% | ___ | 過高>70%? 過低<50%? |
| 夏普比 | ≥0.8 | ___ | 風險調整得當? |
| 最大回撤 | <10% | ___ | 風險控制有效? |
| 盈利因子 | >1.5 | ___ | 優質信號充足? |
| 月回報率 | 1-3% | ___ | 符合預期? |

### 根據結果調整

```
如果勝率↓ → 加強背離確認
如果信號↓ → 放寬成交量要求
如果虧損↑ → 檢查流動性 + 降低倉位
如果波動↑ → 檢查進場點是否精準
```

